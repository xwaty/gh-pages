如何搭建 Hexo 博客

Hexo
网站搭建
Publish Date:   2020-12-14

Hexo 是一个快速、简洁且高效的静态博客框架。

  • 超快速度：Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。
  • 支持 MarkDown：Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。
  • 一键部署：只需一条指令即可部署到 GitHub Pages, Heroku 或其他平台。
  • 插件和可扩展性：强大的 API 带来无限的可能，与数种模板引擎（EJS，Pug，Nunjucks）和工具（Babel，PostCSS，Less/Sass）轻易集成

Hexo 搭建步骤：

1.环境搭建

  • 安装 Git

Win10环境：在页面 https://git-scm.com/download/win 下载并安装

Linux环境：

$ sudo apt-get install git-core

查看版本命令：

$ git --version
  • 安装 Node.js

Win10环境：在页面 https://nodejs.org/en/download/ 下载并安装

Linux环境：

$ sudo apt-get install nodejs
$ sudo apt-get install npm

查看版本命令：

$ node -v
$ npm -v
$ nrm -v

更改 nrm 镜像源：

$ npm i nrm -g
$ set-ExecutionPolicy RemoteSigned(Y)
$ nrm ls
$ nrm use taobao
  • 安装 Hexo
    $ npm install -g hexo
    查看版本命令：
    $ hexo -v

2.搭建网站

$ cd D:\                          # 寻找一个合适的目录
$ hexo init blog                  # 初始化博客
$ cd blog
$ npm install
$ npm install hexo-server --save  # Hexo 3.0 把服务器独立成了个别模块，您必须先安装 hexo-server 才能使用。
$ hexo server                     # 在服务器启动期间，Hexo 会监视文件变动并自动更新
$ hexo s -p 5000                  # 在执行时遇到了 EADDRINUSE 错误，可以在执行时使用 -p 选项指定其他端口

3.撰写博文

$ hexo new 'title'                # 新建一篇博文
$ hexo new draft 'title'          # 新建一篇草稿

在 D:\blog\source_posts 目录中编译内容或直接在 D:\blog\source_posts 目录中新建 .md 文件

$ hexo clean
$ hexo g（ g 代表 generate ）
$ hexo s（ s 代表 server ）

$ hexo s -g                       # 快速更新

4.创建托管仓库

  • 在github上新建repository，命名为：用户名.github.io

  • 在本地项目安装git插件：

    $ npm install --save hexo-deployer-git

  • 修改项目根文件下配置文件：

    $ deploy:
$ 	type: 'git'
$ 	repo: https://github.com/xwaty/xwaty.github.io.git
$ 	branch: master

  • 部署到github上：

    $ git config --global user.email "wwqian@mail.ustc.edu.cn"
$ git config --global user.name "xwaty"

    之后在弹出的页面中输入账号和密码

  • 部署配置

    $ hexo d（ d 代表 deploy ）

5.更换主题

  • 在官网下载相应主题复制到themes文件夹下

或直接在 D:\blog 目录下输入

$ git clone https://github.com/blinkfox/hexo-theme-matery.git themes/matery
$ git clone https://github.com/Siricee/hexo-theme-Chic.git themes/Chic
$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
  • 修改项目根文件下配置文件：
    $ # Extensions
$ ## Plugins: https://hexo.io/plugins/
$ ## Themes: https://hexo.io/themes/
$ theme: 项目名
  • 更新部署
    $ hexo clean
$ hexo g
$ hexo s
$ hexo d
    若出现 “Plugin load failed: %s hexo-theme-landscape” 错误，在 package.json 文件中将最后一行 “hexo-theme-landscape”: “^0.0.3” 删去。
